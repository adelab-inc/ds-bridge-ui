# 01. 프로젝트 개요

> **대상 독자**: 전체 팀 (PM, 디자이너, FE 개발자, AI 개발자)

## TL;DR (핵심 요약)

- **무엇을**: 자연어로 웹 애플리케이션을 생성하는 AI 기반 앱 빌더
- **왜**: 비개발자도 아이디어를 빠르게 프로토타입으로 구현 가능
- **어떻게**: AI가 코드를 생성하고, 실시간으로 프리뷰를 제공
- **참고 서비스**: Lovable, v0.dev, Bolt.new

---

## 프로젝트 비전

### 우리가 만드는 것

사용자가 **자연어로 설명**하면, AI가 **실제 동작하는 웹 앱 코드**를 생성하고,
**즉시 프리뷰**로 결과를 확인할 수 있는 서비스입니다.

```
사용자 입력: "로그인 페이지 만들어줘. 이메일과 비밀번호 입력 필드가 있고,
            소셜 로그인 버튼도 추가해줘"

     ↓ AI 코드 생성

결과: 실제 동작하는 React 컴포넌트 + 실시간 프리뷰
```

### 타겟 사용자

| 사용자 유형 | 니즈 | 기대 가치 |
|------------|------|----------|
| 스타트업 창업자 | 빠른 MVP 제작 | 개발 비용/시간 절감 |
| 디자이너 | 디자인 → 코드 변환 | 개발자 의존도 감소 |
| 프론트엔드 개발자 | 반복 작업 자동화 | 생산성 향상 |
| PM/기획자 | 아이디어 시각화 | 커뮤니케이션 효율화 |

## 핵심 기능

### 1. AI 코드 생성

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   자연어 입력    │ ──→ │    AI 엔진      │ ──→ │   코드 출력     │
│  "버튼 만들어줘" │     │  (Claude API)   │     │  React/TS 코드  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

**특징**:
- 스트리밍 방식으로 코드가 타이핑되듯 생성됨
- React + TypeScript + Tailwind CSS 기반
- 재사용 가능한 컴포넌트 형태로 출력

### 2. 실시간 프리뷰

사용자가 생성된 코드의 결과를 **즉시** 화면에서 확인할 수 있습니다.

**작동 방식**:
- 코드 에디터와 프리뷰 화면이 나란히 표시
- 코드 수정 시 프리뷰가 실시간 업데이트
- 안전한 샌드박스 환경에서 실행

### 3. 코드 에디터

생성된 코드를 직접 수정할 수 있는 전문가용 에디터를 제공합니다.

**기능**:
- 문법 하이라이팅 (코드 색상 구분)
- 자동 완성
- 에러 표시
- 여러 파일 탭 지원

### 4. 프로젝트 내보내기

완성된 프로젝트를 다양한 형태로 내보낼 수 있습니다.

**지원 형태**:
- ZIP 파일 다운로드
- GitHub 저장소 연동
- Vercel/Netlify 원클릭 배포

## 성공 지표 (KPI)

### 제품 지표

| 지표 | 설명 | 목표 |
|------|------|------|
| 코드 생성 성공률 | 오류 없이 실행되는 코드 비율 | > 90% |
| 평균 생성 시간 | 프롬프트 → 프리뷰까지 소요 시간 | < 10초 |
| 사용자 수정률 | 생성 후 수정이 필요한 비율 | < 30% |

## 용어 정의

| 용어 | 설명 |
|------|------|
| **프롬프트** | 사용자가 AI에게 전달하는 자연어 요청 |
| **코드 생성** | AI가 프롬프트를 분석하여 실제 코드를 만드는 과정 |
| **프리뷰** | 생성된 코드가 실제로 어떻게 보이는지 미리 보기 |
| **샌드박스** | 안전하게 격리된 코드 실행 환경 |
| **컴포넌트** | 재사용 가능한 UI 조각 (버튼, 폼, 카드 등) |
| **스트리밍** | 데이터를 한 번에 받지 않고 조금씩 받아 표시하는 방식 |

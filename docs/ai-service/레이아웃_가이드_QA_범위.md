## 검색/필터 영역 – 차이점 분석

> ⚠️ 검색/필터 영역은 현행 기본값과 요청값 간 차이점이 많습니다.

### 1. 현재 코드의 기본 검색/필터 영역 (AI가 인지하는 기본값)

- **12컬럼(`grid-cols-12`) 그리드** 사용, gutter는 `gap-4`(16px)
- 필드는 각자 `col-span-N`(예: `col-span-3`, `col-span-6`)으로 작성
- 필드와 버튼이 한 줄/같은 row에 동시에 존재할 수 있음
- 외부 패딩: `p-6`
- 그리드 컨테이너/아이템에 특별한 내부 마진/패딩 없음

### 2. 요청 내용 (검색/필터 영역 기본값)

- **외부 그리드**: `col-12` 구조, gutter 12px(`gap-3`)
- **카드 내부 패딩**: margin 16px(`p-4`), gutter 12px(`gap-3`)
- **필드 내부**: 별도 서브 그리드(예: `grid-cols-6`) + gutter 12px(`gap-3`)
- **필드 폭**: 6그리드 기반, 각 `col-span-1/2`로 칼럼 너비를 정밀 지정
- **버튼**: 우측 하단 2컬럼, 필드와 별도 행 또는 row 가장 우측 `col-span-2`에 정렬 (별도의 그리드 row 필요)

### 3. 두 디자인의 차이점 (핵심)

| 항목        | 현행 기본값 (AI 인지)                   | 요청값                                                                    |
| ----------- | --------------------------------------- | ------------------------------------------------------------------------- |
| 카드 패딩   | `p-6` (24px)                            | `p-4` (16px)                                                              |
| 그리드 간격 | `gap-4` (16px)                          | `gap-3` (12px)                                                            |
| 필드 구성   | 12컬럼 (예: `col-span-3`, `col-span-6`) | 6컬럼 기반, `col-span-1/2` 또는 12로 환산                                 |
| 버튼 배치   | 간혹 같은 row, 보통 flex로 별도 행      | **필수로 row 하단 우측 2컬럼** 별도 배치 (`grid col-span-2`, row 가장 끝) |
| 서브 그리드 | 없음 또는 1뎁스                         | 필드별 서브 6컬럼 그리드, 내부 gutter=12px                                |

> 즉, 주요 차이는 **gutter와 패딩 단위**, **필드 분할 방법(6컬럼 기준)**, **버튼 위치(별도 row 또는 `col-span-2` 우측 정렬)** 입니다.
> 이 차이에 따라 그리드 구조와 spacing, 버튼 포지션이 달라집니다.

## 액션 버튼 영역 – 우측 정렬 수정

요청에 따라 액션 버튼 영역만 우측 정렬(`flex justify-end`)로 수정되었습니다.

| 항목           | 변경 전                               | 변경 후                              |
| -------------- | ------------------------------------- | ------------------------------------ |
| 정렬 방식      | `flex items-center gap-2` (좌측 정렬) | `flex justify-end gap-2` (우측 정렬) |
| 버튼 종류/순서 | Primary → Secondary → Outline         | 동일                                 |
| 기타 코드/구조 | —                                     | 변화 없음                            |

## 재현: 기본 구조 + 헤더 (섹션 1~3)

| 섹션 | 내용 |
|------|------|
| 1. 기본 구조 원칙 | 해상도(1920px), 그리드 시스템(12col) |
| 2. 전체 레이아웃 구조 | 헤더/메인 섹션 구조, 간격 규칙(Spacing Rule) |
| 3. 헤더 영역 정의 | 글로벌 헤더, 메뉴바, 월드 탭 |

### QA 포인트

- [ ] 해상도/여백/Gutter 수치 정확성 (1920px, 1872px, 24px 등)
- [ ] 간격 규칙 3개 표 값 검증
  - 상위 구조 간격 (헤더↔메인, 타이틀↔콘텐츠)
  - 테이블 구조 간격 (필터바↔그리드, 서머리바↔액션버튼 등)
  - 탭 구조 간격 (탭-그리드, 탭-섹션)
- [ ] 헤더 3단 구조 기능 정의 확인
  - 1단: 글로벌 바 (로고, 통합검색, 바로가기, 사용자 영역)
  - 2단: 메뉴바 (전체 메뉴, 마이메뉴)
  - 3단: 월드 탭 (멀티 탭, 닫기 기능)
- [ ] 이미지 4장 확인 (image.png ~ image 3.png)

## 재민: 메인 섹션 + Grid Type (섹션 4~5)

| 섹션 | 내용 |
|------|------|
| 4. 메인 섹션 구조 | 타이틀 영역, 콘텐츠 영역(검색/필터, 액션버튼, 그리드) |
| 5. Grid Type 정의 | TYPE-A ~ TYPE-H (8개 타입) |

### QA 포인트

- [ ] 타이틀 영역 정의 확인 (메뉴명, 즐겨찾기, 브레드크럼)
- [ ] 콘텐츠 영역 3개 구성요소 검증
  - 검색/필터 영역 (col-12, Gutter 12px, Margin 16px)
  - 액션 버튼 영역 (우측 정렬, 버튼 순서 규칙)
  - 메인 그리드 영역 (AG-Grid)
- [ ] Grid Type 8개 검증

| Grid Type | 컬럼 구조 | 용도 |
|-----------|-----------|------|
| TYPE-A | col-12 | 단일 영역형 |
| TYPE-B | col-6 + col-6 | 균등 2열형 |
| TYPE-C | col-3 + col-9 | 목록+상세형 |
| TYPE-D | col-4 + col-8 | 필터 확장형 |
| TYPE-E | col-4 * 3 | 균등 3열형 |
| TYPE-F | col-2 + col-8 + col-2 | 중앙 집중형 |
| TYPE-G | col-2 + col-2 + col-8 | 탐색 2단계형 |
| TYPE-H | col-3 * 4 | 균등 4열형 |

- [ ] 각 타입별 Variant 정의 확인 (C-1/C-2, D-1/D-2 등)
- [ ] 이미지 12장 확인 (image 4.png ~ image 15.png)

- [ ] Row Pattern 8개 검증

| 코드 | 패턴명 | 목적 |

## 빠른 시작

### 프로젝트 설치

```bash
# 저장소 클론
git clone <repository-url>
cd ds-bridge-ui

# 의존성 설치 (pnpm 사용)
pnpm install

# 개발 서버 실행
pnpm dev
```

### 주요 명령어

| 명령어 | 설명 |
|--------|------|
| `pnpm dev` | 개발 서버 실행 |
| `pnpm build` | 프로덕션 빌드 |
| `pnpm lint` | 코드 린트 검사 |
| `pnpm test` | 테스트 실행 |

## 팀 역할

| 역할 | 담당 영역 |

| **FE 개발자** | UI 컴포넌트, 에디터 통합, 프리뷰 렌더링 |
